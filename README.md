## Como Visualizar

Para visualizar este portfólio localmente, siga os passos:

1.  Clone o repositório:
    ```bash
    git clone https://github.com/alexsabepocoemuito/Portifolio.git
    ```
2.  Navegue até o diretório do projeto:
    ```bash
    cd Portifolio
    ```
3.  Abra o arquivo `index.html` no seu navegador preferido.
